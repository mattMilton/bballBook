Get Started

With Big Red Stats Online Basketball Bookkeeper you are able to easily keep track of all stats in a basketball game. However, this is the premier version of the application, so there are some things that are still in the works. Below is a list of the features we will be adding in future versions.

  • Accounts

    Many wanted features will require user accounts. The ability to save and load frequently used teams is among the most desired. Also, with accounts, recalling saved stats and displaying averages will be enjoyed. Another account based feature will be the ability to grant your friends with accounts permission to observe games you are recording in real time.

  • More Settings

    In game settings, future versions will allow you more detailed control of timeouts (whether certain number per half, or last x number of minutes, etc.). Game date and time will be recorded once game stats are being saved to accounts. In team setup, we are working on allowing you to select a color for each team, as well as selecting which players are starters.

  • Run-time Options

    The plan is to allow you options for run-time. Once these options are in place, we will also implement the auto-stop and auto-start features of the time clock. Currently, you are responsible for stopping and starting the clock. However, the auto-stop and auto-start features will still rely on you to to start the clock much of the time. Because of that, we want to have a different color background and clock when time is stopped to help remind you to start the clock.

  • Scoreboard Changes

    Currently, everything on the scoreboard is accurate to the latest recorded event. When the clock is adjusted, the scoreboard does not reflect the state of the game at the time shown on the scoreboard. This feature is being considered. Also being considered is removing the 'MadeFG' buttons, and the 'Foul' button. The user would then need to click on the team's score or fouls on the scoreboard to initiate the events. This would leave only there or four buttons below the scoreboard.

  • Play Log

    We want to include tabs for each game period in the Plays log in the Game page. You will then be able to look at plays in specific periods, or the entire game. We also would like to include a log of plays in the player detail page. Every play in which the player recorded a stat or a substition would be inclded in their play log.
